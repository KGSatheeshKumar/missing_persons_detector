Help & Instructions

Getting Started

Welcome to the Missing Persons Identification System. This guide will help you understand how to use the system effectively.

Main Features

1. Data Gathering:

Use this feature to add new missing person data to the system. Upload clear face images and provide relevant information.

2. Criminal List:

View and manage the database of known criminals. You can search, filter, and view detailed information about each entry.

3. Start Recognizer:

Upload an image to identify potential matches with known criminals in the database.

4. Missing Person Detect:

Use this feature to identify potential matches for missing persons using facial recognition technology.

Best Practices

  • Ensure uploaded images are clear and well-lit
  • Regularly update the database with new information
  • Double-check results before taking any action
  • Keep the system updated with the latest security patches
